## API Configuration

### Required for Full Functionality

| API | Environment Variable | How to Get | Cost |
|-----|---------------------|------------|------|
| FEC | `FEC_API_KEY` | [api.data.gov/signup](https://api.data.gov/signup) | FREE |
| Linkup | `LINKUP_API_KEY` | [app.linkup.so](https://app.linkup.so) | FREE |
| Senate LDA | `LDA_API_KEY` | [lda.senate.gov/api](https://lda.senate.gov/api) | FREE |

### Optional (Enhanced Features)

| API | Environment Variable | Purpose |
|-----|---------------------|---------|
| Socrata App Token | `SOCRATA_APP_TOKEN` | Higher rate limits (1000/hr vs 60/hr) |
| OpenCorporates | `OPENCORPORATES_API_KEY` | Business registry API (FREE for nonprofits) |
